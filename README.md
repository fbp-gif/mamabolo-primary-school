# Mamabolo Primary School Website

A complete, responsive website for Mamabolo Primary School in the Capricorn District of South Africa. This website redesign maintains all essential information from the original while significantly improving organization, visual appeal, and user experience.

## Features

- **Modern, Responsive Design**: Works perfectly on all devices from mobile to desktop
- **Complete School Information**: All original content preserved and better organized
- **Teacher Assignment Details**: Clear presentation of staff roles and subject assignments
- **Interactive Elements**: Booking system, donation forms, and dynamic content
- **Accessibility Features**: Keyboard navigation, semantic HTML, and ARIA labels
- **Performance Optimized**: Fast loading with minimal external dependencies

## Website Structure

1. **Homepage**: School overview with key statistics and welcome message
2. **About**: School history, mission, and vision
3. **Academics**: Detailed curriculum by phase (Foundation, Intermediate, Senior)
4. **Staff**: Complete teacher listings with subject assignments
5. **Calendar**: 2026 academic calendar and public holidays
6. **Governance**: School Governing Body (SGB) information
7. **Contact**: Location, map, and tour booking
8. **Donations**: Support options and secure donation processing
9. **Resources**: Useful links for parents, teachers, and students

## How to Run

### Option 1: CodeSandbox (Recommended)
1. Go to [codesandbox.io](https://codesandbox.io)
2. Create a new sandbox with HTML/CSS/JS
3. Copy the three files (index.html, style.css, script.js) into the sandbox
4. The website will automatically render in the preview pane

### Option 2: Local Computer
1. Create a folder named `mamabolo-primary-school`
2. Save the three files (index.html, style.css, script.js) in this folder
3. Open `index.html` in any modern web browser
